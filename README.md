# Clock using Pure JavaScript

A simple **analog clock** built using **HTML, CSS, and JavaScript**.
The clock updates every second and rotates the hour, minute, and second hands based on the current system time.

## 🚀 Features

* Real-time clock using JavaScript `Date()` object
* Smooth rotation of hour, minute, and second hands
* Responsive clock design using CSS
* Built using pure JavaScript (no libraries)

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript

## 📂 Project Structure

```
Clock-using-pure-JS
│
├── index.html

├── script.js

```

## ⚙️ How it Works

JavaScript runs a `setInterval()` function every second
